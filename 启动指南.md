# AI笔记应用启动指南

## 快速启动（推荐）

1. **双击运行 `start.bat` 文件**
   - 会自动启动后端和前端服务
   - 后端地址：http://localhost:8000
   - 前端地址：http://localhost:5173

## 手动启动

### 1. 启动后端

```bash
cd ai-notes-backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 启动前端（新开一个终端）

```bash
cd ai-notes-frontend
npm install
npm run dev
```

## 功能说明

- **用户注册/登录**：基本的用户认证功能
- **创建笔记**：添加新的读书笔记
- **AI生成摘要**：点击"AI生成摘要"按钮，使用SiliconFlow模型自动生成笔记摘要
- **查看笔记列表**：浏览所有已创建的笔记

## API密钥配置

应用已内置测试用的SiliconFlow API密钥。如需使用自己的密钥，请设置环境变量：

```bash
set SILICONFLOW_API_KEY=your-api-key-here
```

## 注意事项

- 确保Python和Node.js已安装
- 首次运行需要安装依赖包
- 数据库文件会自动创建在 `ai-notes-backend/notes.db`
